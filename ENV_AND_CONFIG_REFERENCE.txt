Rahul Prints Configuration Reference
===================================

This project no longer uses Firebase.

1. Copy `.env.example` to `.env`
2. Fill in your own values
3. Start the app with `npm start`

Required values
---------------

`BUSINESS_EMAIL`
Owner email address that should receive every order PDF.

`BUSINESS_PHONE`
Phone number shown in the UI and in generated invoices.

`RAZORPAY_KEY_ID`
Razorpay publishable key used by the browser checkout.

`RAZORPAY_KEY_SECRET`
Razorpay secret used only on the server to create orders and verify payment signatures.

`UPI_ID`
Optional legacy UPI fallback. Razorpay is the primary customer checkout flow.

Choose one email provider
-------------------------

For local testing or your own mailbox, use SMTP:

`MAIL_PROVIDER=smtp`
Uses any SMTP account, including Gmail, Zoho, Outlook, Brevo, or your own domain mailbox.

`SMTP_SERVICE`
Optional shortcut for well-known providers such as `gmail`, `outlook365`, or `zoho`.

`SMTP_HOST`
Optional direct SMTP server hostname when your provider gives host and port details.

`SMTP_PORT`
Optional SMTP port, for example `465` or `587`.

`SMTP_SECURE`
Optional boolean. Set `true` for implicit TLS, usually port `465`, or `false` for STARTTLS, usually port `587`.

`SMTP_USER`
SMTP login used to send invoices.

`SMTP_PASS`
SMTP password or app password for the account above.
If you use Gmail, use an App Password instead of your normal Google password.

For Render or other hosts that block SMTP, use Mailjet:

`MAIL_PROVIDER=mailjet`
Uses Mailjet's HTTPS API instead of SMTP.

`MAIL_FROM_EMAIL`
Verified sender address shown in the From header.
Avoid free Gmail, Yahoo, Outlook, or Hotmail addresses with Mailjet, because deliverability may fail.
Keep `BUSINESS_EMAIL` and `ORDER_NOTIFICATION_EMAIL` as your real inbox if needed, and use a verified custom-domain sender here instead.

`MAIL_FROM_NAME`
Display name shown to customers.

`MJ_APIKEY_PUBLIC`
Mailjet API key.

`MJ_APIKEY_PRIVATE`
Mailjet secret key.

Optional values
---------------

`WHATSAPP_PROVIDER=callmebot`
Enables automatic WhatsApp alerts to your own WhatsApp number.

`WHATSAPP_PHONE`
Your WhatsApp number in international format, for example `+919345574203`.

`CALLMEBOT_API_KEY`
API key received from CallMeBot after authorizing your number.

`PRICE_BW`
Price per black and white page. Default: `1`

`PRICE_COLOR`
Price per color page. Default: `10`

`PAYMENT_TIMEOUT_MINUTES`
Visible payment timer for the Razorpay checkout session. Default: `5`

`DRY_RUN_NOTIFICATIONS=true`
Optional local testing mode. Generates the invoice and returns success without sending real emails or WhatsApp messages.

Security note
-------------

The old Firebase file contained a Gmail app password directly in source code.
That password should be rotated immediately in Google Account settings if it was ever used.
